# RogueGame
A single-player game in C using the ncurses library!

## Run
Compile with:
`-l ncurses` 

## Play
* Use 'w', 'z', 'a', and 's' to move up, down, left, and right, respectively. <br />
* Collect all gold ('G') and avoid all monsters ('M'). <br />
* Hit 'q' at any time to quit. <br />
